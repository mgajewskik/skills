# Feedback and Practice

Use this whenever the user asks for critique, review, coaching, or wants to improve their own writing.

## Goal

Do not stop at "here is the better version." Teach the user what changed, why it works, and how to repeat it.

## Feedback format

Return feedback in this order:

### Keep
- 1-2 strengths worth preserving
- name the exact strength, not a generic compliment

### Change
- 3-5 highest-leverage issues
- tie each issue to a phrase, sentence pattern, or structure choice
- explain the fix in plain language

### Practice
- one habit to build
- one short drill the user can do in 2-5 minutes
- one checkpoint to use before sending the next draft

## High-value coaching categories

1. Clarity
- unclear pronouns
- buried thesis
- vague nouns or weak verbs

2. Concision
- filler lead-ins
- repeated meaning
- nominalizations and corporate fog

3. Structure
- slow openings
- missing BLUF
- paragraphs doing too much at once

4. Tone and audience fit
- too formal, too casual, too vague, too defensive, too promotional

5. Learnable habits
- one sentence, one job
- put the ask first
- replace abstract nouns with verbs
- read aloud and cut the sentence where you run out of breath

6. Validation
- test whether a reader can paraphrase the point and next action after one read
- test whether deadlines, owners, and decisions are explicit

## Drill library

Choose one drill that matches the main issue.

- BLUF drill
  - Rewrite the first two sentences so the ask or point appears by sentence two.

- Verb drill
  - Circle every form of "is/are/was/were" and every noun ending in `-tion`, `-ment`, or `-ance`. Replace at least three with direct verbs.

- Specificity drill
  - Replace three vague words such as `thing`, `important`, `soon`, `a lot`, or `better` with a concrete noun, date, number, or measurable claim.

- Paraphrase drill
  - After the rewrite, restate the main message and next action in one sentence without looking. If you cannot, rewrite the opening.

- Voice drill
  - Mark three phrases that sound like you. Keep them in the rewrite unless they block clarity.

- Read-aloud drill
  - Read the draft aloud once. Split or simplify every sentence that makes you stumble.

## Feedback-only mode

If the user asks for feedback without a rewrite:
- do not rewrite the full text unless useful
- quote small excerpts
- show one or two model rewrites for the biggest problems
- focus on patterns the user can reuse

## Coaching tone

- be direct, calm, and specific
- do not praise vaguely
- do not shame the user for rough drafts
- assume the user can improve quickly with the right pattern
